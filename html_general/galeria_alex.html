Este es un codigo hecho en python con la extencion de pygames para hacer el juego clasico tetris, en la imagen se muestra como se imprime la puntuacion, nivel y velocidad. Tiene un sistema de random para que las piezas preestablecidad aparezcan al asar y asi puedas irlas acomodando.

Este es el juego clasico "Breakout", se puede apreciar que tiene un contador de vidas, cada bloque tiene una puntuacion distinta y la pelota va aumentando de velocidad conforme choca con los bloques. Tiene un diseño basico de colores solidos vividos dando un aspecto retro y que anima a jugar.

Esta imagen el del juego Pong hecho en python con ayuda de pygame, se tiene un marcador que va contando cuantos puntos tiene cada jugar al tocar el lado de su contrincante. Se mantiene un aspecto de blancvo y negro para mantener la fidelidad de pong orginal.
